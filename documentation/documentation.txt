Our main.py file will be used to execute the program

Responsibilities have been split using the MVC method

For this we have created 3 folders each have differing Responsibilities
    1. Model - Manages data and logic
    2. View - Display data
    3. Controller - Handles input, refreshes view and updates model

Additionally for easier routing each folder contains a __init__.py file which allows python to 
recognize the folder as a package this allows for easy importing into main using . method
